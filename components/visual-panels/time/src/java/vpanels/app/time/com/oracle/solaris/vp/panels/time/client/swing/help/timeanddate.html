Time and Date Settings

After selecting “Time & Date” in the Time selector pane, you can change the date or time by selecting the value that needs to be changed and using the appropriate arrows to increment or decrement the value. As you change the date, a monthly calendar for that date is displayed. You can also change the date using the widgets in the displayed calendar. You can also change the time by manually moving the hands on the clock.
