The Service Firewall Panel

The Service Firewall panel is used to configure system-wide or service-specific firewall policy settings. The top of the Service Firewall panel includes two pieces of information. The Status field on the left shows if the service is enabled or not. The User field on the right shows the credential that is being used by the tool. If you click on the lock button on the far right, you can change the credentials. See Changing User Credentials for more information about changing the user credentials.

The panel displays different tabs depending on the firewall policy that is selected in the left hand list:

  • Default Settings – From this screen, the settings for all services enabled on the system can be set. See Default Settings for the System-Wide Firewall Policy for more information.

  • Open Program Settings – From this pane, ports and protocols can be configure to allow incoming traffic. See Open Programs Settings for the System-Wide Firewall Policy for more information.

  • Override Settings – This tab is used to configure policies that override service specific policies that are set. See Override Settings for the System-Wide Firewall Policy for more information.
