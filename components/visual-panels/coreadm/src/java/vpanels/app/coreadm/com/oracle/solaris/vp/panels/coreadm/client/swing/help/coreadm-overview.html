Core Files Panel

The Core Files Panel can be used to configure the names, locations and contents of core files that are generated by the system. To write scripts to manage core files, see the coreadm(1M) man page.

The top of the Core Files panel includes two pieces of information. The Status field on the left shows if the service is enabled or not. The User field on the right shows the credential that is being used by the tool. If you click on the lock button on the far right, you can change the credentials. See Changing User Credentials for more information about changing the user credentials.

The panel includes the following components:

  • Core Files Tab – This tab lists all of the core files that are on the system. See Core File Selection for more information.

  • Settings Tab – This tab allows you to view or edit the core file generation settings. See Settings for Core Files for more information.
