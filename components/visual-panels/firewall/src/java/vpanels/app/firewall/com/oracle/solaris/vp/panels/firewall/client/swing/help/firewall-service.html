Individual Service Firewall Policies

From the Firewall Policies selector, you can select a service-specific firewall policy to configure. The service-specific firewall policy take precedence over the default system-wide policies, but not the system-wide override policies. For each service, you can select to:

  • Use the default system-wide access policy

  • Deny or allow access from a specific host, network or interface. In particular for the selected service, you can:

    • Add a host, network or interface to the access policy

    • Edit the entries in the access policy

    • Delete the entries from the access policy

    • Add, edit or delete exceptions to the access policy

  • Choose not to enforce an access policy
