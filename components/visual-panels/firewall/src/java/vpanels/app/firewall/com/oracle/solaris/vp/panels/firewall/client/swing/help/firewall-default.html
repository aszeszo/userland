Default Settings for the System-Wide Firewall Policy

The following operations are supported on the Default tab:

  • Enable the host-based firewall service

  • Set the system-wide policy to deny or allow access from a specific host, network or interface.

    • Add a host, network or interface to the policy

    • Edit the entries in the policy

    • Delete the entries from the policy

    • Add, edit or delete exceptions to the access policy

  • Choose not to enforce any access policy

  • Choose to use a custom policy file

The system-wide policies that are set in this tab can be changed in the policy for each individual service.
