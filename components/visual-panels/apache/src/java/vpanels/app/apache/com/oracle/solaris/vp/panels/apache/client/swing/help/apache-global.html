Global Settings for the Apache Web Service

The Global Settings pane can be used to:

  • Enable or disable the Apache web service.

  • Identify the location of a complete configuration file for the web server. This option overrides any configuration information that may be added to the global settings pane. See Why Would I Use a Custom Configuration File with the Apache Server? for more information.

  • Specify which MIME types are defined for files on all local virtual hosts. For more information, see What are MIME Types in the Apache Service?.

  • Specify which Apache modules are available for use by all local virtual hosts. For more information, see What are Apache Modules?.

For more information see The Apache Web Server Panel.
