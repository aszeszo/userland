The Time and Date Settings Panel

The Time and Date Settings panel can be used to set the time, the date, and the time zone. Also you can select to use the Network Time Protocol (NTP) service.

At the top of the panel are two pieces of information. The Status field on the left shows how the date is currently managed. The User field on the right shows the credential that is being used by the tool. If you click on the lock button on the far right, you can change the credential. See Changing User Credentials for more information about changing the user credentials.

The Time Selector pane on the right allows you to select:

  • Time & Date – Allows you to change the date and time. See Time and Date Settings for more information.

  • Time Zone – Allows you to change the time zone, See Time Zone Settings for more information.

  • Network Time – Allows you to configure NTP to manage time settings.
